# Como apresentar no portfólio

1. Abra a landing e explique a tese: proposta como experiência, não PDF feio.
2. Login demo → dashboard com dados reais de seed.
3. Mostre criação por brief e a sala pública.
4. Aceite + CTA de pagamento.
5. Mostre limite do Starter e a página `/app/upgrade`.
6. Abra `src/billing/` e explique adapters + webhook idempotente.
7. Feche com roadmap Fase 2 e monetização.
