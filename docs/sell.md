# Como vender a primeira versão

1. Ofereça onboarding feito com o cliente (primeira proposta no ar em 30 min).
2. Comece no Starter (marca ProposalRoom) e faça upsell para Pro quando houver 4ª proposta ativa.
3. Pitch: “sua proposta vira landing page com aceite e pagamento”.
4. Cobrança: Pro R$89 / Business R$249.
5. Canal: agências locais, designers, consultorias e assistência técnica.
6. Prova social: 3 cases demo no seed + PDF/CSV export.
